변수를 사용하는 이유
- 데이터 저장: 변수는 데이터를 저장하는데 사용되어 프로그램이 특정 데이터를 기억하고 활용할 수 있음
- 유지보수성: 변수를 사용하면 데이터의 변경이 필요한 경우 한 곳에서만 수정하여 전체 코드에 적용할 수 있음
- 가독성 향상: 변수를 사용하면 코드의 가독성이 향상되며, 의미 있는 이름을 사용함으로써 코드를 이해하기 쉽게 만듦
- 재사용성: 변수를 통해 동일한 값이나 데이터를 여러 곳에서 사용할 수 있어 코드를 재사용하기 용이함

변수 선언과 할당
 - 변수를 선언하기 위해서는 let 또는 const 키워드 사용
    let: 변수를 선언하고 값을 재할당할 수 있는 키워드
    const: 변수를 선언하고 초기에 값을 할당한 후에는 값을 변경할 수 없는 키워드

기본 데이터 타입 (원시 타입)
 - 주로 사용되는 기본 데이터 타입은 숫자, 문자열, 불린 세 가지이다.
    숫자: 정수 또는 소수점이 있는 수
    문자열: 텍스트로 이루어진 데이터
    불린: 참(true) 또는 거짓(false) 값을 가지는 데이터

- 그리고 숫자, 문자열, 불린 외에도 다음과 같은 데이터 타입이 있음
    null: 값이 없음을 나타내는 특별한 값
    undefined: 할당되지 않은 변수의 기본 값
    bigint: 큰 정수를 나타내는 데이터 타입 (정수가 2^53 -1 보다 클 경우)
    symbol: 고유하고 변경 불가능한 값 (주로 객체 속성의 키로 사용되며 충돌을 피하기 위해 생성됨)

돔 (DOM) 모델이란 무엇인가?
- DOM은 웹 페이지의 구조를 표현하고, JavaScript를 사용하여 페이지의 내용을 동적으로 변경할 수 있는 인터페이스를 제공    
- DOM은 웹 페이지의 HTML 문서를 계층 구조로 표현하며, 각 요소는 노드로 표현됨, DOM을 통해 JavaScript는 HTML 요소에 접근하고 조작할 수 있음

DOM 요소 선택, 조작 예제
getElementById 외에도 다양한 방법으로 DOM 요소를 선택하고 조작할 수 있음
    querySelector: CSS 선택자를 사용하여 첫 번째 일치하는 요소를 반환
    querySelectorAll: CSS 선택자와 일치하는 모든 요소를 NodeList로 반환
    getElementByClassName: 클래스 이름으로 요소를 선택할 수 있음
    getEleemntByTagName: 태그 이름으로 요소를 선택할 수 있음

이벤트 처리(핸들링)
- 웹 애플리케이션에서 사용자의 상호 작용을 감지하고 대응하는 핵심 부분
- 이벤트는 웹 애플리케이션에서 발생하는 사용자 동작이나 브라우저의 다양한 상태 변화를 의미함
- '클릭', '마우스 오버', '키보드 입력', '로딩 완료' 등 다양한 종류의 이벤트가 있음
